== Description ==

A plugin that just adds an admin button for clearing the PHP OPcache if the server enabled it. That's it.

== Installation ==

1. Upload the OPcache Clear plugin to to your website
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Find a button in the top admin bar (only if your server has OPcache enabled)
